## Findings

### JTBD-ADMIN-STUDENT-01 — Create a new student end-to-end
- **Persona:** Ibu Nur
- **Completed:** no
- **Severity:** blocker
- **Observation:** As Ibu Nur I navigate to `/admin/students` from the sidebar. The page header "Akademik / Siswa" renders and a "Siswa" `<h1>` exists in DOM, but the entire main content region (search box, "Tambah Siswa" button, the 5-row student table) sits inside `<div hidden id="S:1">` and never becomes visible. After 15+ seconds the screen remains blank below the breadcrumb. There is nothing for me to click — `Tambah Siswa` has zero bounding-box dimensions because its hidden ancestor is `display:none`. I cannot start a new student.
- **Evidence:** `document.querySelectorAll('main')[1].innerText.length === 0` while `document.querySelectorAll('tbody tr').length === 5`; the rows live inside `<div hidden id="S:1">` (Next.js Suspense placeholder). `GET /api/students?limit=5` returns 200 with the expected payload — server data layer healthy.
- **Suggestion:** Investigate why the inline `$RC` dehydration script for Suspense boundary `S:1` (and `S:0` on dashboard) is not flipping the placeholder from `hidden` → visible on staging. Likely a Next.js 16 streaming regression, an asset-chunking mismatch on Vercel, or a CSP / nonce error blocking the inline `$RC(…)` calls.
