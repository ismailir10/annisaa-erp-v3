## Summary

User instruction lifted the 6-job cap to cover every teacher module + page; all 13 JTBDs in `docs/uat/jobs/teacher.md` were exercised. Five blockers and four majors surfaced. The dominant pattern is **client-side route navigation that takes 3–6 s wall-clock to settle**: tab switches, sheet opens, accordion expands, and PDF generation all crossed strict thresholds. Two structural gaps are more concerning than the timings: profile photo upload is **not implemented at all** (persona cannot complete PROFILE-01), and the salary-slip page has **no in-app detail view** (rows are not tappable; only PDF download is offered, and the PDF is A4-landscape so does not fit a single mobile screenshot). Class-attendance, own-attendance, and assessment-scoring data flows are all functionally correct — autosave fires, `POST /api/student-attendance/mark` returns 200, the "✓ Tersimpan" badge confirms scoring saves. RSC prefetch GETs to `/teacher/assessments` returned **HTTP 503** during the run; subsequent first-class navigations succeeded but the 503 noise on the same surface is worth checking against Vercel function logs.

## Heuristic disclaimer
This is synthetic UAT. An LLM persona cannot replicate thumb reach, sunlight
glare, emotional distrust, or network conditions on a 4-year-old phone in
Bekasi traffic. Treat findings as a cheap first pass, not a substitute for
putting the app in front of a real teacher or parent.

In addition: this run was driven through Chrome MCP on a desktop browser at 1568 × 755 px against staging on Vercel. Real Bu Sari uses a mid-range Android in portrait (≈375 px wide) on intermittent 4G — so the desktop run **understates** layout friction (no horizontal-scroll evidence on the slip PDF, for example, even though portrait users would see it) and **overstates** click latency (desktop ethernet is faster than 4G; thresholds are tuned for 4G, so the breaches above are real and would be worse on phone). GPS could not be granted to the Chrome MCP-controlled tab; the GPS-denial path was exercised instead.
