Open question · from Ismail

How should a parent with 2–4 kids switch and monitor them?

Today's pill-tabs are thin: they scale badly past 2 kids, hide per-child state (is Yusuf's invoice also unpaid?), and force a tap-to-see-anything loop. Islamic courtesy layer

An Nisaa' is an Islamic PAUD/TKIT. Arabic greetings and honorifics matter — they signal respect to the community. But over-use feels performative and clutters transactional UI. Rules:

Use
  • Assalamu'alaikum — parent dashboard only, once per session
  • Ustadz / Ustadzah — teacher greeting; leave-approval notice from admin; parent → teacher notes
  • Bismillah — optional on onboarding splash or new-academic-year banner
  • InsyaAllah — pending-approval copy ("InsyaAllah akan diproses dalam 1×24 jam")
  • Alhamdulillah — payment success toast for parent
  • Bapak / Ibu + first name — admin → parent correspondence (email, SMS)
Avoid
  • Arabic greetings in every page header — once per session is enough
  • Ustadzah on every toast / button — dilutes the honorific
  • Forcing Arabic in admin back-office — "Selamat datang" is fine
  • Religious framing around money ("rezeki", "berkah") — transactional stays neutral
  • Gendered child nouns ("putra Anda") — use the child's name
  • InsyaAllah on hard deadlines (due dates, payroll cut-off) — sounds evasive

Glossary · canonical terms

Use
  • Siswa — student (never "murid" in UI)
  • Karyawan — employee (not "pegawai")
  • Wali / Wali Murid — guardian ("orang tua" OK in copy)
  • Ustadz / Ustadzah — teacher honorific
  • Tagihan — invoice (not "invois")
  • Jatuh Tempo — due / overdue
  • Nonaktifkan / Aktifkan — deactivate / activate
  • Hadir · Tidak Hadir · Sakit · Izin — 4 attendance labels
  • Periode Penggajian — payroll period
  • Tahun Ajaran — academic year (not "tahun akademik")
  • Rapor — report card (not "laporan nilai")
  • Kampus — campus
  • Program — program tier (TK A, TK B, KB)
Avoid
  • Delete → Nonaktifkan (soft delete only)
  • Submit → Simpan / Kirim / domain verb
  • Click here → inline verb link
  • OK / Cancel → Ya, … / Batal
  • Are you sure? → describe the action + side-effect
  • Error / Success as toast body → describe what
  • Data alone → name the entity
  • User → Pengguna, or role (Guru, Admin)
  • Login / Logout → Masuk / Keluar
  • Putra / Putri → child's name + "ananda"
  • Rezeki / berkah around money — stays transactional
Writing mechanics → Indonesian sentence case for titles (not Title Case). Currency Rp 1.250.000 with non-breaking space. Dates: 12 Okt 2025 in tables, Minggu, 12 Oktober 2025 in headers. Time: 24-hour, tabular-nums. Never abbreviate child names in official docs — always full name in rapor, invoice, slip.
