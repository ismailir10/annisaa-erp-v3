Parent Portal — Cycle 4 Prototype (v2 — teacher-DNA hybrid)

v1 was overfitted to journal section-bar mechanism. v2 hybridizes: teacher portal light-card pattern + eyebrow labels + focal moments per page (action surfaces — home, invoices, reports), journal week-grid + sticky col + today lane (data-grid surface — attendance). STAGING banner removed (not part of design). Page identity restored (h1 every page).

Tokens — none new required

  • All frames use existing globals.css tokens (status triad, celebration gold, primary teal, type/space scale).
  • Patterns reused from existing components: EmptyState (warm + celebration accents), Card (light), PageHeader (calm), portal week-grid (journal exact).
