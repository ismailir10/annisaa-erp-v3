## Implementation

- **FIND-012** — `app/admin/invoices/page.tsx` — pass `onCreated={() => { fetchInvoices(); fetchStats(); }}` to `<ManualInvoiceDialog>`. Pre-fix the prop was omitted and the dialog navigated straight to detail without warming the list cache; the return trip rendered the loading skeleton for 3-5s.
- **FIND-015** — `app/teacher/home-client.tsx` — added a `mounted` state flag flipped inside the existing live-clock `useEffect`. Time-derived rendering (`timeStr`, `dateStr`, `greeting`) now produces empty/placeholder strings on the server render and the real values appear after hydration. Eliminates the SSR↔client text-content mismatch (React error #418) when no AttendanceRecord exists for today.
- **FIND-007** — `app/admin/(hr)/employees/page.tsx` — after `fetchPositions()` returns, if the array is empty, populate a default preset list `["Guru Kelas","Guru Pendamping","Kepala Sekolah","Admin Sekolah"]` so a fresh-tenant admin sees actionable Jabatan options. Inline-add path unchanged.
- **FIND-011** — `app/admin/admissions/page.tsx` — extracted the one-shot stats fetch into a `fetchStats` callback and bound the existing `useEffect` to it. Every admission mutation handler (`convertToStudent`, `handleSubmit`, `advanceStatus`, `handleCancel`) now calls `fetchStats()` next to its `fetchAdmissions()` so the top KPI cards no longer go stale until F5.
- **FIND-013** — `app/admin/page.tsx` — `last7Weekdays` builder now seeds today's date first (when it's a weekday), then walks back. Pre-fix the cursor decremented before push, so today was silently excluded from the 7-weekday window — on a fresh staging DB with attendance rows only for today, the panel rendered "Data kehadiran belum tersedia" despite real data.
- **FIND-004** — `app/api/config/campuses/route.ts` GET accepts optional `?status=ACTIVE|INACTIVE|ALL` (defaults ACTIVE). `app/admin/settings/campuses/page.tsx` renders a status filter pill above the grid; inactive view swaps the row kebab to a single "Aktifkan kembali" action that PUTs `{ status: "ACTIVE" }`. Empty state copy adjusted per-filter.
